GroupAdmin module for Drupal 6.x

original developer: netgenius
contact:  drupal(a)netgenius.co.uk
support:  Commercial support is available from the developer -- www.netgenius.co.uk

Project page on drupal.org: http://drupal.org/project/groupadmin


Description

Provides a simple administrative interface to manage Organic Groups members, allowing users to be searched by name or email address and added to or removed from a group.


See http://www.netgenius.co.uk/node/72 for documentation.

VERSION 2 BETA
==============

* If you are upgrading from version 1, yor configuration will not be retained - go to (site)/admin/og/groupadmin to configure the module.  The same is true if you downgrade from version 2 to version 1.

* Support for Content Profile is currently not available - user version 1 if you need that.
