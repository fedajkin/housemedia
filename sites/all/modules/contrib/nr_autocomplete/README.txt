Description
-----------
This module provides a new widget for the nodereference CCK field type.

Features
--------
* Single autocomplete input for nodereference field with multiple values
* Works similarly to taxonomy's free-tagging autocomplete (comma-separated)

Prerequisites
-------------
* content.module (main CCK module)
* nodereference.module (included with CCK)

Installation
------------
Copy the nr_autocomplete folder to the appropraite modules folder, such as
sites/all/modules, then enable the module in Site Building -> Modules.
